# Oasis Landing

Upcoming landing page for Oasis.

[oasis-landing-staging.now.sh](https://oasis-landing-staging.now.sh/)

## Development

Start the development server:

`yarn develop`

Build:

`yarn build`

Update the staging website:

`now --prod`

(You can install "now" with `yarn global add now`)
